Synchronizes wordpress users with MailChimp email marketing service.

== Description ==

Synchronizes wordpress users with MailChimp. Select MailChimp list, group and subgroup (subscribers organized in a subgroup or according to your role), and automatically new users are added to this mailchimp group.
Add, update and delete synchronized users.


Official site: [eggemplo](http://eggemplo.com/plugins/users2mailchimp/)

== Installation ==

You can either install it automatically from the WordPress admin, or do it manually:

1. Unzip the archive and put the `users2mailchimp` folder into your plugins folder (/wp-content/plugins/).
1. Activate the plugin from the Plugins menu.

== Frequently Asked Questions ==

= What options using the sync button? =

It take the general options, except "Need confirm" option, which is always set to YES.

= Where is setup page ? =

In "Users" group menu. 
Users -> Users2MailChimp 

= Where I get the api key? =

In your Mailchimp account menu -> API keys & Authorized Apps
